#  Heroku

## 概述

用于在 Heroku 上部署 ws。

**Heroku 为我们提供了免费的容器服务。**

**可以部署两个以上的应用，实现 [负载均衡](https://toutyrater.github.io/app/balance.html)。**

**Heroku 的网络并不稳定**
